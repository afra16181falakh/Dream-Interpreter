#  DreamBot – Your AI Dream Decoder 

Ever wake up like "Whoa, what *was* that dream about?"  
Well, DreamBot’s got you. 
This Gen Z-friendly AI buddy interprets your dreams and helps uncover what your subconscious might be tryna tell you — powered by Google Generative AI.

---

##  What Is DreamBot?

DreamBot is an AI-powered tool that analyzes user-inputted dreams and generates insightful interpretations. It taps into psychological, symbolic, and emotional layers to provide personalized meaning behind your nighttime adventures.

> Think of it like therapy… but for your dreams 

---

## Features

-  Dream interpretation with personality  
-  Conversational feedback on emotional themes  
-  Insight into symbols, settings, and characters  
-  Purpose-driven analysis (aka "why did my brain do that?")  
-  Built using `google-generativeai==0.3.2`  

---

##  Requirements

Install the required package:
```bash
pip install google-generativeai==0.3.2

Also make sure:
	•	You’re using Python 3.7+
	•	You’ve got a Google Generative AI API Key

⸻

⚙ Setup
	1.	Clone the repository:
git clone https://github.com/your-username/dreambot.git
cd dreambot
	2.	Create a .env file and add your API key:
GOOGLE_API_KEY=your_google_api_key_here

	3.	Run the bot:
python dreambot.py
